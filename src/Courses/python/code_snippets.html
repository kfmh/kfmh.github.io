Linux or Mac
Create an alias to exicute actions in sequence by adding this at the top or bottom of your .zshrc or .bashrc file:

The example code bellow exceutes:

  1. Create a new python3 environment named .env, in current directory
  2. pip install requirements.txt
  3. Activate environet

github.com/kfmh

#Syntax: 
#alias new-command="command-to-exicute"
alias p3env="python3 -m venv .env \
    && pip install -r requirements.txt \
    && source .env/bin/activate"
